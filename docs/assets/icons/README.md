# TSiJUKEBOX - Ícones das Seções

Este diretório contém os 8 ícones modernos criados para representar as principais seções da documentação e interface do TSiJUKEBOX.

---

## Visão Geral

Todos os ícones seguem o **Design System** do projeto:
- **Fundo:** Preto escuro (#0a0a0a)
- **Estilo:** Neon com efeito glow
- **Formato:** PNG de alta resolução
- **Tamanho médio:** ~5 MB cada

---
